## Problem Statement
This project solves the problem of face mask detection. Given an image of a person, the model predicts whether the person is wearing a mask or not.

## Dataset
The dataset is sourced from Kaggle and contains around 10,000 images across two classes — mask and no mask.

## Model
The model is built using a CNN-based deep learning architecture implemented in TensorFlow/Keras.

## Results
It achieved a validation accuracy of 97-98%.

## How to Run
Download the dataset from Kaggle, unzip it, and it's ready to use.
